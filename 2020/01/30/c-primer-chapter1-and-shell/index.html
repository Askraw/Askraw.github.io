c++primer chapter1 and shell

目前在重读C++Primer，因为是新版大缘故也新加了一些C++11的特性，在看书之外会把需要实践的问题和课后习题标注，统一编程实践。

Shell Script

首先shell脚本在ubuntu系统是没有后缀名的。通过直接创建文件编写然后修改其权限，之后即可运行复用。

接下来是关于shell脚本的一些语法习惯的问题，常用的编译和插件跟定制化的功能相比还是不够懂人心，所以考虑到长期的使用和发展，目前我的想法就是通过脚本的方式去进行的。通过一定的命名规则来保证整体的一致性和使用的便捷性。

本脚本目前还非常简陋未来可以不断的完善和充实丰富。它现在的作用就是以一个给定的命名方式，通过读入章节和问题的数字来编译和运行cpp程序。

C++ Primer Chapter 1

对于cppprimer的第一章的内容，我觉得似乎对于已经了解过C++的人来说有点奇技淫巧了。虽然以目前我的水平说这个话并不算合适，但是我希望我能在未来的工作中以好的习惯来减少出现这种刁钻问题的可能，精力能更集中到更重要的层面的问题和方式上来。换句话说，第一章里出现的问题，对于规范的编程者来说是写不出这样的代码的，这样的话如果花时间去琢磨这些问题对于25岁的我，一个即将工作的人来说并不合适。
